Eso me dio la confianza para sobreponerme a lo que serían los últimos semestres de la carrera.

Conceptos

Función:
Una función en JavaScript es un bloque de código reutilizable que realiza una tarea específica. Puedes definir funciones para dividir tu código en secciones más pequeñas y más manejables. Una función se define con la palabra clave function.

let:
Se utiliza en Javascript para declarar variables. Las variables declaradas con tienen un alcance limitado al bloque o la función en la que se declaran. Esto significa que no se pueden acceder fuera de ese bloque.

const:
Se usa para la declaración de variables en JavaScript. A diferencia de let, las variables declaradas con const son inmutables, lo que significa que su valor no puede cambiar después de la inicialización. Además, también tienen un alcance de bloque.

Etiqueta HTML "div"
Se utiliza para crear un contenedor genérico en HTML. Es una etiqueta de bloque que se utiliza para agrupar y estructurar contenido en secciones de la página web. Los "div" son ampliamente utilizados para aplicar estilos y organizar elementos en el diseño de la página.

Etiqueta HTML "form"
se utiliza para crear un formulario en HTML. Los formularios permiten a los usuarios ingresar datos que pueden ser enviados al servidor para su procesamiento. Contienen campos de entrada, botones y otros elementos relacionados con la recopilación de información.

Etiqueta HTML "input"
Se utiliza en el contexto de un formulario y define un campo de entrada de datos. Puede tomar varias formas, como campos de texto, casillas de verificación, botones de radio y más. Los usuarios pueden ingresar datos en estos campos o seleccionar opciones específicas.
